# 정의
입력을 (-1, 1) 범위로 압축하는 활성화 함수.
$$\tanh(x) = \dfrac{e^x - e^{-x}}{e^x + e^{-x}}=2\sigma(2x)-1$$
[[Sigmoid]]를 선형 변환한 형태이며, 출력이 0을 중심으로 대칭이라는 점에서 은닉층 활성화 함수로 Sigmoid보다 선호된다.

# 특성
- 출력 범위: (-1, 1)
- 미분: $\tanh'(x) = 1 - \tanh^2(x)$. 최댓값은 $x=0$일 때 1.
- 원점대칭: 출력의 평균이 0이므로 다음 레이어의 입력 분포가 편향되지 않음.
- Sigmoid와 마찬가지로 포화 구간에서 기울기가 0에 수렴해 [[Vanishing Gradient]] 문제가 발생한다.
> [!note] 출력 범위의 차이
> Sigmoid는 (0, 1)이라 항상 양수이고, Tanh는 (-1, 1)이므로 음수도 나올 수 있다. 원점대칭이 좋은 이유가 여기서 나온다. 은닉층 출력이 전부 양수면 다음 레이어의 gradient가 항상 같은 부호로만 흐르게 되므로, 학습이 지그재그로 느려진다. Tanh는 이 문제를 해결한다. 

# 대표 변형 / 관련 기법
- [[Sigmoid]]: 출력 범위 (0, 1)인 상위 함수.
- [[ReLU]]: 포화 문제를 구조적으로 해결한 대안
- [[LSTM]]: cell state 계산에 Tanh를 사용.
