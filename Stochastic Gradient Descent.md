# 정의
손실함수 $L$을 최소화하기 위해 파라미터 $\theta$를 기울기 반대 방향으로 반복적으로 업데이트하는 최적화 알고리즘. 전체 데이터셋 대신 무작위로 샘플링한 미니배치(혹은 단일 샘플)를 사용하여 기울기를 근사한다.
$$\theta \leftarrow \theta - \eta \cdot \nabla_\theta L(\theta;x^{(i)}, y^{(i)})$$
- $\eta$: 학습률 (learning rate)
- $x^{(i)}$, $y^{(i)}$: 무작위 샘플링된 학습 예제

# Batch Gradient Descent와의 비교
|         | Batch GD |            SGD            |
| ------- | :------: | :-----------------------: |
| 기울기 계산  |  전체 데이터  |        단일/미니배치 샘플         |
| 업데이트 빈도 |  에폭당 1회  |           샘플마다            |
| 노이즈     |    없음    | 있음 (local minima 탈출에 유리함) |
| 계산 비용   |    높음    |            낮음             |

# 핵심 특징
- 대규모 데이터셋에서 Batch Gradient Descent보다 훨씬 빠르게 수렴한다.
- 기울기 노이즈가 local minima 탈출에 유리하게 작용하기도 한다.
- 학습률 스케줄: [[Word2Vec]]에서는 초기 학습률 0.025에서 학습 종료 시 0으로 선형 감소한다.

# 파생 옵티마이저
- [[AdaGrad]]: 파라미터별로 학습률을 다르게 적용한다. 희귀한 파라미터에 더 큰 업데이트를 가한다.
- [[RMSProp]]: Adagrad의 학습률 소멸 문제를 지수 이동 평균으로 완화했다.
- [[Adam]]: 모멘텀 + RMSProp을 결합한 옵티마이저로, 현재 가장 널리 사용된다.
- SGD + Momentum: 이전 업데이트 방향을 누적하여 수렴 속도를 향상시켰다.
